• Get started fast!

To see Zoomify in action immediately, simply double-click the TemplateWebPage.htm file in this download or use this link. Your web browser will open and instantly display the image ZoomifyImageExample. Use the mouse to click-zoom or click-drag, or use the Toolbar or Navigator window to explore!

• Three easy steps to Zoomify your high-quality images!

1. Drag your image file onto the Zoomify Converter to rapidly convert it into a Zoomify Image folder. (You can ignore the temporary file "zoomifyConvertPrefs.asp" that is also created each time the converter runs. Feel free to delete it if you like.)

2. Edit the TemplateWebPage.htm file in any HTML or text editor. Change the text "ZoomifyExampleImage" to the name of your Zoomify Image folder. Be sure to change this text in two places in the file (the first is for Windows IE users, the second is for everyone else). Save the file.

3. Copy your Zoomify Imagefolder, your edited TemplateWebPage.htm file, and the ZoomifyViewer.swf file to your Web server - that's it - the web page is ready for your site visitors to view your image!

For more detailed steps and extensive demos and information visit the Zoomify Support page!

• Display parameters

Zoomify Express supports HTML parameters for specifying the path to the image to display as well as whether to show or hide the Navigator (bird's-eye-view). The '&' character is used to combine parameters.

Use '&' to delimit parameters. For example:

   zoomifyImagePath=myFolder/myImage&zoomifyNavigatorVisible=0

Note that 0 hides the Navigator, while 1 shows it. An example web page name TemplateWebPage.htm is included in Zoomify Express. View or change its HTML by opening it in any HTML or text editor.

Many additional HTML and XML parameters are supportd by the Zoomify Design product available from the Zoomify web site. And the Zoomify Flash product includes complete source code for Flash editors. For more information please visit the product demos, support, and sales pages of the Zoomify web site.
